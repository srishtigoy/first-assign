Web Developer

It contains two dedicated developers or performed by one specialist:

Front-end developers:

They deal with the client-side of the website using such technologies as CSS, HTML, and JavaScript. Their principal task is to make sure your website looks good on any device.

Back-end developers:

They are responsible for the creation of databases and content management system. There are a lot of languages and frameworks for this purpose (JavaScript, php, SQL etc).

Full-stack developers:

These developers can build both a client-side and server-side of the project on their own.
